# SkillSphere

**SkillSphere** is a high-performance educational platform designed for modern professionals to master new technologies. Built with **Next.js 16 (Turbopack)** and **Better Auth**, the application focuses on providing a seamless, secure, and interactive learning experience.

## Live: [skill-sphere-himel.vercel.app](https://skill-sphere-himel.vercel.app/)
---

### 🚀 Key Features

- **Secure Authentication**: Robust session management using Better Auth and social login via Google.
- **Interactive Course Discovery**: Real-time course searching and filtering powered by Framer Motion.
- **Smooth UI/UX**: High-end animations including staggered reveals, continuous ticker-tape carousels using Swiper.js, and skeleton loading states.
- **Responsive Design**: A clean, minimalist interface optimized for both desktop and mobile users.
- **Performance First**: Leverages Next.js App Router and Turbopack for lightning-fast development and production speeds.

---

### 🛠️ Tech Stack

- **Framework**: Next.js 16
- **Styling**: Tailwind CSS & DaisyUI
- **Animations**: Framer Motion & Swiper.js
- **Auth**: Better Auth
- **Forms**: React Hook Form
- **Database**: MongoDB

---

### 📦 Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/0xW1zard/SkillSphere.git
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3. **Run**
    ```bash
    npm run dev
    ```
